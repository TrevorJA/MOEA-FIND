# Stage 05 — Multi-site MOEA-FIND (DRB)

## Purpose

Apply the MOEA-FIND method to the Delaware River Basin multi-site
problem with co-located Kirsch generators across the DRB headwater
gauges. Backs §7.1-7.2 of the manuscript and Fig 8.

## Run order

Must run **after** stage 02 (constraint calibration). Outputs feed into
stage 06 (Pywr-DRB re-evaluation consumes the multi-site Pareto front)
and stage 99 (manuscript figure aggregation).
