# Stage 01 — Analytic validation

## Purpose

Validate the Manhattan-norm trick on a synthetic objective space where the
true Pareto manifold is known analytically. Backs the manuscript's §5
analytic-validation section and the K=1..6 dimension sweep that motivates
applying MOEA-FIND to constrained hydrologic problems where uniform space
filling is non-trivial. Contributes Figs 1, 2, 3, 4 (main) and SI-1.

## Run order

No prerequisites. This stage is the entry point and runs in a bare
Python environment (no SynHydro). Run order within the stage is
independent — each driver writes to its own `outputs/expNN_*/` slug.
