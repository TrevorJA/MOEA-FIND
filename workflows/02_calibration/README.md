# Stage 02 — Calibration

## Purpose

Pre-requisite calibration runs that produce the JSON tolerance files
consumed by stages 03–05. This stage also produces the SI figures that
justify each calibration choice (constraint tolerances, DV-space
uniformity tolerance, wrapper geometry, Kirsch convergence).

## Run order

Must run **before** stage 03 and stage 04. The DV-space uniformity
ablation (stage 04) depends on `dv_uniformity_calibration.py`; the
hydrologic-constraint arm depends on `constraint_calibration.py`.
Stage 04 runs are gated on the JSON outputs of those two scripts.

## Manuscript figures

This stage's outputs back the SI calibration subsections (§SI-A through
§SI-E). No main-text figures originate here. Working PDFs land in
`figures/calibration/`; final assets are regenerated by
`workflows/99_manuscript_figures/make_figures.py` into
`figures/supplementary/`.

## SI figure budget

Each subsection (§SI-A..§SI-E) targets 2-3 multi-panel figures:

- §SI-A Constraint calibration: tolerance histograms + acceptance ROC.
- §SI-B DV-uniformity calibration: bootstrap distribution of L2*/KS/AD.
- §SI-C Wrapper fidelity + geometry: marginal preservation, mapping smoothness.
- §SI-D Kirsch convergence: wall-clock vs trace count.
- §SI-E Metric historical blocks: per-metric block distributions.
